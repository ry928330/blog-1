### 快捷 git 命令查阅

```bash
cat ~/.oh-my-zsh/plugins/git/git.plugin.zsh
```

一些常用命令:

```
ga . & gcmsg '..' & gp
gst === git status
gco === git checkout
gcb === git checkout -b
gb === git branch
```
